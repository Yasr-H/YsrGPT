# YsrGPT: Academic Policy Advisor

YsrGPT is a private, local RAG (Retrieval-Augmented Generation) system designed to act as an Academic Advisor for Iqra University students. It uses **Ollama** and **Streamlit** to provide instant answers from official policy manuals without sending data to the cloud.

## Features
- **100% Local:** No APIs, no data leaks.
- **Academic Specialist:** Trained on Iqra University's Academic Policy Manual.
- **Bilingual:** Supports English and Roman Urdu.
